# Canto (canto)

Canto is a digital asset management (DAM) platform offering a fully documented REST API for organizing, tagging, distributing, and sharing brand assets including images, videos, and documents across teams and channels. The API uses OAuth 2.0 for authentication and provides endpoints for asset search, metadata management, folder and album operations, automated workflows, and asset distribution.

## APIs

### Canto REST API

The Canto REST API enables developers to connect Canto media libraries with other applications. It supports asset search, metadata management, folder and album operations, automated workflows, and asset distribution. Authentication uses OAuth 2.0 with support for both authorization code and client credentials grant types.

- **Human URL:** [https://www.canto.com/api/](https://www.canto.com/api/)
- **Base URL:** `https://api.canto.com/`
